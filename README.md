# Lightbox Gallery

A simple image gallery with a lightbox effect.

You can click on any image to open it in a lightbox view, then navigate between images or close the lightbox.

## Features

- Click image to open lightbox
- Next and previous image navigation
- Keyboard controls (Arrow Left, Arrow Right, Escape)
- Close lightbox by clicking outside or on close icon
- Responsive layout using Bootstrap

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Font Awesome

## How to Run

Open the `index.html` file in your browser.

## Controls

- **Next Image:** Right Arrow or next icon
- **Previous Image:** Left Arrow or previous icon
- **Close Lightbox:** Escape key or click outside the image
